python-cine
===========

Reading Vision Research cine files with python

## Requirements
- numpy
- opencv

### On OSX
You might need to install [Homebrew](http://brew.sh/) first. 
```
brew install python opencv
```

```
pip install -r requirements.txt
```


### On Ubuntu
```
apt-get install python-opencv python-numpy
```

```
pip install -r requirements.txt
```
